ðŸ‘‰ **[Systematic Analysis of the openYuanrong Distributed Computing Engine]({{ site.baseurl }}/conference-activities/eurosys-2026/openYuanrong/)**  
This tutorial will introduce the openYuanrong distributed computing engine. openYuanrong is dedicated to building a set of serverless architecture distributed applications. It provides a multi-language function programming interface, simplifying distributed application development with a single-machine programming experience. It also offers capabilities such as distributed dynamic scheduling and data sharing, enabling high-performance operation of distributed applications and efficient resource utilization in clusters.
